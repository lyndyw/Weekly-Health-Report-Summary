# 健康管理周总结报告

用于生成每周健康管理总结报告。

## 功能定位
- 一周健康监测数据汇总
- 血糖 / 血压 / 体重达标情况统计
- 行为依从性回顾
- 与上周对比分析
- 下周建议

## 推荐输入
- 客户姓名、周次、时间范围
- 空腹/餐后血糖列表
- 血压列表
- 体重起止值
- 饮食、运动、用药、症状记录
- 如需对比，请提供上周数据

## 脚本说明
- `scripts/generate_report.py` 可以静态通过语法检查
- 但脚本内容混合了“单周周报”和“9 周整合报告”逻辑，定位不够单纯
- 建议优先把它作为参考实现，不直接作为唯一正式输出来源

## 推荐输出
1. 基本信息
2. 核心指标变化
3. 症状与体征
4. 行为与依从性
5. 本周亮点
6. 下周建议

## 示例提示词
- 请帮我生成第 3 周健康管理周报
- 这周血糖血压和体重数据都在这里，帮我总结本周亮点和下周建议
- 对比上周，看看这一周哪里改善了
