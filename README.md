# 健康管理周报生成器

🩺 专业慢病管理专家 — 根据用户健康数据自动生成结构化调理周报告

## 安装

本技能已预装在 OpenClaw 工作区，无需额外安装。

## 使用方法

### 方式 1：命令行交互

```bash
cd /root/.openclaw/workspace/skills/health-weekly-report
python3 scripts/run.py
```

然后按提示输入健康数据。

### 方式 2：JSON 数据文件

```bash
python3 scripts/generate_report.py examples/liu-yuan-week1.json
```

### 方式 3：OpenClaw 集成

在 OpenClaw 中直接说：
```
生成调理报告
客户姓名：张三
第几周：3
空腹血糖：5.2, 5.8, 5.3, 6.1, 5.5, 5.9, 5.4
餐后血糖：8.2, 9.1, 7.8, 10.2, 8.5, 7.9, 8.1
血压：128/82, 132/85, 125/80, 135/88, 130/83, 127/81, 129/82
干预前体重：65kg
干预后体重：64.2kg
...
```

## 输入数据格式

### 必要字段

| 字段 | 类型 | 示例 |
|------|------|------|
| 客户姓名 | 字符串 | `张三` |
| 第几周 | 数字 | `3` |
| 空腹血糖 | 数字列表 | `[5.2, 5.8, 5.3, 6.1, 5.5, 5.9, 5.4]` |
| 餐后血糖 | 数字列表 | `[8.2, 9.1, 7.8, 10.2, 8.5, 7.9, 8.1]` |
| 血压 | 二维列表 | `[[128,82], [132,85], ...]` |
| 干预前体重 | 数字 | `65.0` |
| 干预后体重 | 数字 | `64.2` |

### 可选字段

| 字段 | 说明 |
|------|------|
| 上周体重 | 用于计算累计减重 |
| 水肿情况 | 症状记录 |
| 自我感觉 | 症状记录 |
| 尿量 | 症状记录 |
| 尿泡沫 | 症状记录 |
| 饮食执行 | 依从性记录 |
| 运动情况 | 依从性记录 |
| 用药情况 | 依从性记录 |
| 监测习惯 | 依从性记录 |
| 饮食日志 | 用于归因分析 |

## 输出示例

```markdown
# 张三第 3 周调理总结

## 1. 核心指标变化情况

### 1.1 血糖变化情况

| 干预目标 | 干预结果 | 原因 |
|----------|----------|------|
| 空腹血糖 3.9-7.0 mmol/L | 7 次达标 | 饮食控制良好 |
| 餐后 2 小时 <10.0 mmol/L | 6 次达标，1 次超标 | 周三晚餐吃了红薯 |

...
```

## 干预目标配置

可在 `scripts/generate_report.py` 中修改 `TARGETS` 配置：

```python
TARGETS = {
    "血糖": {
        "空腹": {"min": 3.9, "max": 7.0, "unit": "mmol/L"},
        "餐后 2 小时": {"max": 10.0, "unit": "mmol/L"}
    },
    ...
}
```
