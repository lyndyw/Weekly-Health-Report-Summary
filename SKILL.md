# 健康管理周报生成器

> 🩺 专业慢病管理专家 — 自动生成结构化调理周报告

## 快速开始

### 触发方式

用户发送以下任一触发词 + 健康数据：
- "生成调理报告"
- "周报"
- "健康总结"
- "本周调理情况"
- "生成第 X 周调理总结"

### 输入数据格式

```markdown
客户姓名：张三
第几周：3
监测数据：
- 血糖：空腹 [5.2, 5.8, 5.3, 6.1, 5.5, 5.9, 5.4]，餐后 [8.2, 9.1, 7.8, 10.2, 8.5, 7.9, 8.1]
- 血压：[128/82, 132/85, 125/80, 135/88, 130/83, 127/81, 129/82]
- 体重：干预前 65kg，干预后 64.2kg
- 饮食日志：[每日饮食记录]
- 运动情况：[每日运动记录]
- 用药情况：[每日用药记录]
- 症状记录：[水肿/自我感觉/尿量等]
```

### 输出

生成结构化 Markdown 报告，包含：
1. 核心指标变化（血糖/血压/体重）
2. 症状与体征记录
3. 行为与治疗依从性回顾
4. 本周亮点与进步
5. 本周知识分享与下周建议

---

## 报告结构

### 1. 核心指标变化情况

#### 1.1 血糖变化情况

| 干预目标 | 干预结果 | 原因 |
|----------|----------|------|
| 空腹血糖 < 7.0 | 7 次达标 | 饮食控制良好 |
| 餐后 2 小时 < 10.0 | 6 次达标，1 次超标 | 周三晚餐吃了红薯 |

**总结及建议：**
- 空腹血糖控制稳定
- 建议调整晚餐主食结构

#### 1.2 血压变化情况

| 干预目标 | 干预结果 | 原因 |
|----------|----------|------|
| 收缩压 ≤ 130 | 4 次达标，3 次超标 | 睡眠不好、心情波动 |
| 舒张压 ≤ 80 | 5 次达标，2 次超标 | 盐摄入偏高 |

**建议：**
- 控盐技巧：出锅前放盐
- 情绪调节：多聊天诉说

#### 1.3 体重变化情况

| 指标 | 干预前 | 干预后 | 变化幅度 |
|------|--------|--------|----------|
| 体重 | 65.0kg | 64.2kg | -0.8kg ↓ |
| BMI | 25.3 | 25.0 | -0.3 ↓ |

**建议：**
- 体重平稳下降，有助于减轻心肾负担
- 继续保持当前饮食节奏

### 2. 症状与体征记录

| 观察项目 | 本周情况 | 趋势变化 |
|----------|----------|----------|
| 水肿情况 | 下肢水肿明显改善 | → 好转 |
| 自我感觉 | 睡眠改善，精力提升 | 整体向好 |

### 3. 行为与治疗依从性回顾

| 项目 | 执行情况 | 评价 |

| 监测习惯 | 每日监测血糖血压 | ★★★★★ 优秀 |

### 4. 本周亮点与进步

1. **体重管理成功** — 下降 0.8kg，有助于减轻心肾负担
2. **监测习惯养成** — 连续 7 天完整记录数据
3. **饮食控制改善** — 主动减少高糖食物摄入

### 5. 本周知识分享与下周建议

## 配置

### 干预目标（可自定义）

```json
{
  "血糖": {
    "空腹": {"min": 3.9, "max": 7.0, "unit": "mmol/L"},
    "餐后 2 小时": {"max": 10.0, "unit": "mmol/L"}
  },
  "血压": {
    "收缩压": {"max": 130, "unit": "mmHg"},
    "舒张压": {"max": 80, "unit": "mmHg"}
  },
  "体重": {
    "每周目标下降": {"min": 0.5, "max": 1.0, "unit": "kg"}
  }
}
```

### 报告模板

模板文件位于 `templates/report-template.md`，可根据需求调整格式。
